# CS405 Security Portfolio

## 📌 Overview

This repository contains my work for CS 405, focused on secure coding practices, vulnerability prevention, and applying real-world cybersecurity concepts to software development.

The goal of this portfolio is to demonstrate how security should be integrated throughout the development process instead of being added at the end.

---

## 🔐 Project One – Security Policy

This project includes a complete security policy for Green Pace, covering:

* Defense in Depth
* Least Privilege
* Secure Coding Standards
* Risk Assessment and Mitigation
* Encryption (in transit, at rest, and in use)
* Triple-A Framework (Authentication, Authorization, Accounting)

This project shows how security principles can be applied at both the system and development level.

## 🎤 Project Two – Security Presentation

This section includes a presentation and script explaining key cybersecurity concepts such as:

## 💻 Coding Activities

* Encryption implementation

These projects focus on identifying vulnerabilities early and applying secure coding standards.
